# 1. Installing Node.js

## What is Node.js?

Node.js is a runtime environment that allows JavaScript to run outside the browser.

Originally, JavaScript and TypeScript were designed to run only inside web browsers. Node.js made it possible to execute JavaScript on a computer/server environment, enabling developers to build many different types of applications.

Since Playwright supports JavaScript and TypeScript, Node.js is required to run Playwright tests.

---

## Download Node.js

Search for:

```text
Node.js
```

Open the official website:

```text
https://nodejs.org
```

---

## Installation Options

Node.js provides two installation approaches:

### Option 1 — Install Using NVM

NVM (Node Version Manager) allows managing multiple Node.js versions.

Recommended for:

- Advanced users    
- Developers working with multiple Node.js versions

---

### Option 2 — Prebuilt Installer (Recommended)

This is the simpler and recommended option for most users.

Download the installer based on:

- Operating System    
    - Windows
    - macOS
    - Linux
- Processor Architecture
    - x64 → Intel/AMD processors
    - ARM64 → Apple Silicon / Qualcomm chips
    - x86 → Older 32-bit systems

---

## Important Note

Always install the **LTS (Long Term Support)** version of Node.js.

### Avoid Installing

- Current version    

because it may contain:

- Experimental features
- Compatibility issues
- Instability with frameworks/tools

---

## Installation Process

The installation process is straightforward:

1. Download installer    
2. Open installer
3. Click:
    - Agree
    - Continue
    - Install
4. Finish installation

# 3. Installing Git

## Check if Git is Already Installed

Open:

- Terminal    
- Command Prompt

Run:

```bash
git -v
```

or

```bash
git --version
```

Example output:

```bash
git version 2.43.0
```

If you see a version number, Git is already installed.

---

## Installing Git on Windows

Search for:

```text
git scm
```

Open:

```text
https://git-scm.com
```

Download:

- Git for Windows x64    

Install normally.

---

## Installing Git on macOS

For macOS, Git installation is usually done using **Homebrew**.

---

## Install Homebrew

Run the installation command from the Homebrew website inside the terminal.

Example:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

After installation:

- Execute the additional terminal commands shown by Homebrew    
- This adds Homebrew to the system path

---

## Install Git Using Homebrew

Run:

```bash
brew install git
```

---

## Validate Git Installation

Run:

```bash
git --version
```

---

# 5. Install Playwright VS Code Extension

## Steps

1. Open VS Code    
2. Go to Extensions panel
3. Search for:

```text
Playwright
```

4. Install:

```text
Playwright Test for VSCode
```

---

## Benefits of the Playwright Extension

The extension allows you to:

- Run tests    
- View test results
- Debug tests
- Manage projects
- Use Playwright Test Explorer
- Improve development workflow

After installation, the Playwright test runner icon appears in VS Code.
