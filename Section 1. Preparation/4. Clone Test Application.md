## Introduction

In this lesson, we will download and run the test application that will be used throughout this Playwright course.

The application will help us practice automating different UI components and scenarios.

---

# 1. Open GitHub Repository

Navigate to:

```text
https://github.com
```

Search for:

```text
pw-practice-app
```

Open the repository:

```text
Artem-PW/practice-app
```

---

# 2. Copy Repository URL

Click:

```text
Code
```

Copy the repository clone URL.

Example:

```bash
https://github.com/Artem-PW/practice-app.git
```

---

# 3. Clone the Repository

Choose a folder on your computer where you want to keep the project.

Example:

```text
Desktop/demo
```

Open terminal inside that folder.

Run:

```bash
git clone https://github.com/Artem-PW/practice-app.git
```

After cloning, a new project folder will be created.

Example:

```text
practice-app
```

---

# 4. Open Project in VS Code

Inside VS Code:

```text
File → Open Folder
```

Select the cloned project folder.

Example:

```text
Desktop/demo/practice-app
```

Click:

```text
Open
```

The project is now opened in VS Code.

---

# 5. Install Project Dependencies

Open a new terminal inside VS Code.

Run:

```bash
npm install --force
```

This command installs all required project dependencies.

---

## Notes About Warnings

During installation, you may see:

- Dependency warnings    
- Conflict warnings
- Vulnerability warnings

These are usually acceptable for this course.

### Important

Only worry if you see actual installation errors.

Warnings are generally fine.

---

# 6. Start the Application

Run:

```bash
npm start
```

Wait for the application to build.

Example successful message:

```text
Compiled successfully
```

The application will usually run at:

```text
http://localhost:4200
```

---

# 7. Open the Application in Browser

Open browser and navigate to:

```text
http://localhost:4200
```

You should now see the test application running.

---

# Application Features

This sample application contains various UI components such as:

- Input fields    
- Forms
- Date pickers
- Dialog boxes
- Checkboxes
- Radio buttons    
- Dropdowns

We will automate these components throughout the course using Playwright.

---

# 8. Stopping the Application

Inside terminal:

Press:

```text
CTRL + C
```

This stops the application server.

If you refresh the browser after stopping the server, the application will no longer be available.

---

# 9. Restart the Application

To run the application again:

```bash
npm start
```

Wait until you see:

```text
Compiled successfully
```

Refresh the browser and the application will work again.

---

# Important Note About Terminals

Keep the application terminal running while working with Playwright.

### Recommended Setup

## Terminal 1

Used for running the application:

```bash
npm start
```

---

## Terminal 2

Used for:

- Playwright commands    
- Running tests
- Other terminal operations

This allows both processes to run simultaneously.
