In this lesson, we learn how to automate **date pickers in Playwright**, select a specific date from a calendar UI, and validate the selected value in an input field.

---

## Scenario Overview

We will:

1. Navigate to the **Datepicker page**
2. Open the calendar input field
3. Select a specific date from the calendar
4. Validate that the selected date appears in the input field

---

## Step 1: Navigate to Date Picker Page

```typescript
test("Date Picker (Part 1)", async ({ page }) => {
  await page.getByText("Forms").click();
  await page.getByText("Datepicker").click();

  const calenderInputField = page.getByPlaceholder("Form Picker");

  await calenderInputField.click();
});
```

---

## Step 2: Open the Calendar

We locate the input field using its placeholder:

```typescript
const calenderInputField = page.getByPlaceholder("Form Picker");
```

Then click it to open the calendar:

```typescript
await calenderInputField.click();
```

---

## Step 3: Selecting a Date from the Calendar

When inspecting the calendar UI, each date is represented by:

```
day-cell ng-star-inserted
```

So we can select a date using:

```typescript
await page
  .locator('[class="day-cell ng-star-inserted"]')
  .getByText("14")
  .click();
```

---

## Important Challenge: Hidden Month Values

The calendar includes days from **previous and next months**.

For example:

- May dates (28–31) appear inside June calendar view    
- Selecting them may give incorrect results

So we must ensure we select only the **current month dates (June)**.

---

## Step 4: Handling Partial Match Issue

If we try:

```typescript
.getByText("1")
```

Playwright may match:

- 1    
- 10
- 11
- 12
- etc.

This leads to incorrect selection.

---

## Step 5: Fix using Exact Match

We solve this using:

```typescript
{ exact: true }
```

Example:

```typescript
await page
  .locator('[class="day-cell ng-star-inserted"]')
  .getByText("1", { exact: true })
  .click();
```

---

## Step 6: Full Working Test

```typescript
test("Date Picker (Part 1)", async ({ page }) => {
  await page.getByText("Forms").click();
  await page.getByText("Datepicker").click();

  const calenderInputField = page.getByPlaceholder("Form Picker");

  await calenderInputField.click();

  await page
    .locator('[class="day-cell ng-star-inserted"]')
    .getByText("1", { exact: true })
    .click();

  await expect(calenderInputField).toHaveValue("Jun 1, 2026");
});
```

---

## Step 7: Assertion

We validate that the selected date is correctly reflected in the input field:

```typescript
await expect(calenderInputField).toHaveValue("Jun 1, 2026");
```

