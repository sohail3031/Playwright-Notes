## Scenario Overview

We will test the **Age filter feature** in the Smart Table.

We will:

1. Enter different age values in the filter input  
2. Validate that the table only shows rows matching that age  
3. Handle the special case where no data is found (e.g., age = 200)

## Step 4: Initial full implementation

```ts
test("Web Tables (Part 2)", async ({ page }) => {
  await page.getByText("Tables & Data").click();
  await page.getByText("Smart Table").click();

  const ages = ["20", "30", "40", "200"];

  for (let age of ages) {
    await page.getByRole("textbox", { name: "Age" }).clear();
    await page.getByRole("textbox", { name: "Age" }).fill(age);

    await page.waitForTimeout(500);

    const ageRows = page.locator("tbody tr");

    for (let row of await ageRows.all()) {
      const cellValue = await row.locator("td").last().textContent();
      expect(cellValue).toEqual(age);
    }
  }
});
```

---

## Problem: No data for age 200

When `age = 200`, the table shows:

```
No data found
```

So validation fails because:

- No rows exist
- Loop cannot run correctly

---

## Step 5: Handle empty result case

```ts
if (age == "200") {
  expect(await page.getByRole("table").textContent())
    .toContain("No data found");
}
```

---

## Final Working Test

```ts
test("Web Tables (Part 2)", async ({ page }) => {
  await page.getByText("Tables & Data").click();
  await page.getByText("Smart Table").click();

  const ages = ["20", "30", "40", "200"];

  for (let age of ages) {
    await page.getByRole("textbox", { name: "Age" }).clear();
    await page.getByRole("textbox", { name: "Age" }).fill(age);

    await page.waitForTimeout(500);

    const ageRows = page.locator("tbody tr");

    for (let row of await ageRows.all()) {
      const cellValue = await row.locator("td").last().textContent();

      if (age == "200") {
        expect(await page.getByRole("table").textContent())
          .toContain("No data found");
      } else {
        expect(cellValue).toEqual(age);
      }
    }
  }
});
```
