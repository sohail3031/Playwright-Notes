# JavaScript vs TypeScript

JavaScript and TypeScript are very similar.

### TypeScript

TypeScript is essentially:

- A superset of JavaScript    
- JavaScript with type support

We will begin with JavaScript fundamentals first and later discuss the differences between JavaScript and TypeScript.
