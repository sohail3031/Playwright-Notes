In this lesson, we learn about two important JavaScript concepts:

- Concatenation    
- Interpolation

Both are used to combine strings and variables together to create dynamic messages.

---

# Concatenation

## What is Concatenation?

Concatenation means combining multiple strings and variables together using the `+` operator.

---

## Example Scenario

Imagine we are building a shopping cart website and want to display a message like:

```javascript
"The price for your Cup is 50 dollars"
```

The product name (`Cup`) and price (`50`) can change depending on the item being purchased.

So instead of hardcoding values, we use variables.

---

## Creating Variables

```javascript
var price = 50
var itemName = "Cup"
```

- `price` stores the product price    
- `itemName` stores the product name

---

## Creating a Message Using Concatenation

```javascript
var messageToPrint = "The price for your " + itemName + " is " + price + " dollars"

console.log(messageToPrint)
```

### Output

```javascript
The price for your Cup is 50 dollars
```

---

## Changing Variable Values

```javascript
var price = 80
var itemName = "Table"

var messageToPrint = "The price for your " + itemName + " is " + price + " dollars"

console.log(messageToPrint)
```

### Output

```javascript
The price for your Table is 80 dollars
```

---

## How Concatenation Works

Concatenation breaks a string into smaller parts and combines them using the `+` operator.

Example:

```javascript
"Hello " + name + "!"
```

- Strings and variables are joined together    
- The final result becomes a single string

---

# Interpolation

## What is Interpolation?

Interpolation is another way to insert variables directly inside a string.

Instead of using `+`, JavaScript uses:

- Template literals    
- Backticks `` ` ``
- `${}` syntax

---

## Syntax of Interpolation

```javascript
`${variableName}`
```

---

## Example

```javascript
var messageToPrint2 = `The price for your ${itemName} is ${price} dollars`

console.log(messageToPrint2)
```

### Output

```javascript
The price for your Table is 80 dollars
```

---

# Important Notes

## Backticks

Interpolation uses **backticks** instead of double quotes.

Example:

```javascript
`This is a template literal`
```

The backtick key is usually located:

- Below the `Esc` key    
- Left of the number `1` key

---

# Concatenation vs Interpolation

| Concatenation                 | Interpolation                                   |
| ----------------------------- | ----------------------------------------------- |
| Uses `+` operator             | Uses `${}` syntax                               |
| Multiple strings are combined | Variables are inserted directly into the string |
| Older approach                | Modern JavaScript approach                      |
| Can become difficult to read  | Cleaner and easier to read                      |

---

# Example Comparison

## Concatenation

```javascript
var message = "Hello " + userName + ", your score is " + score
```

## Interpolation

```javascript
var message = `Hello ${userName}, your score is ${score}`
```

---

# Summary

- Concatenation combines strings using the `+` operator    
- Interpolation uses template literals and `${}`
- Both approaches create dynamic strings
- Interpolation is generally cleaner and easier to read
- Both methods are commonly used in JavaScript

> [!note]  
> Interpolation is often preferred in modern JavaScript because it improves readability and makes long strings easier to manage.
