## Introduction

Welcome to the JavaScript Fundamentals module.

In this lesson, we will create a simple **Hello World** application using JavaScript and Node.js.

Creating a Hello World application is a traditional first step when learning a new programming language.

---

# Prerequisites

Before starting this lesson, make sure your development environment is already configured:

- Node.js installed
- VS Code installed

If not, please go through:

[[3. Configuration of Development Environment]]

---

# 1. Create Project Folder

Create a new folder on your computer.

Example:

```text
JS Fundamentals
```

---

# 2. Open Folder in VS Code

Open Visual Studio Code.

Navigate to:

```text
File → Open Folder
```

Select the newly created folder.

Example:

```text
JS Fundamentals
```

Click:

```text
Trust the Authors
```

The folder should now open in VS Code.

At this point, the project is empty.

---

# 3. Initialize Node.js Project

Open terminal inside VS Code:

```text
Terminal → New Terminal
```

Run:

```bash
npm init
```

---

## What is `package.json`?

During initialization, npm creates a configuration file called:

```text
package.json
```

This is the main configuration file for a Node.js project.

It contains:

- Project information
- Scripts
- Dependencies
- Metadata

---

## Accept Default Values

npm will ask several setup questions.

You can simply press:

```text
Enter
```

multiple times to accept all default values.

At the end, confirm with:

```text
yes
```

A new `package.json` file will be created.

---

# 4. Create Project Structure

Create a new folder:

```text
Lessons
```

Inside the `Lessons` folder, create a new JavaScript file:

```text
Lesson1.js
```

---

# 5. Write Hello World Program

Add the following code inside `Lesson1.js`:

```javascript
console.log("Hello World!");
```

---

# Understanding `console.log()`

`console.log()` prints output to the terminal/console.

In this example, it prints:

```text
Hello World!
```

---

# 6. Run the JavaScript File

Open terminal and navigate to the `Lessons` folder:

```bash
cd Lessons
```

---

## Run the File

Use the following command:

```bash
node file_name.js
```

Example:

```bash
node Lesson1.js
```

---

# Output

You should see:

```text
Hello World!
```

---

# Folder Structure

Your project structure should now look like this:

```text
JS Fundamentals/

│

    └── Lesson1.js
```
