In this lesson, we learn about logical operators in JavaScript.

Logical operators are used to combine multiple conditions and return a boolean result:

- `true`
- `false`

---

# Types of Logical Operators

| Operator | Name        |
| -------- | ----------- |
| `&&`     | Logical AND |
| \|\|     | Logical OR  |
| `!`      | Logical NOT |

---

# Logical AND (`&&`)

## What is Logical AND?

The logical AND operator combines two or more expressions.

The result is `true` only if **all conditions are true**.

---

# Syntax

```javascript
console.log(true && true)
```

### Output

```javascript
true
```

Explanation:

- Left side is `true`    
- Right side is `true`
- Both are true, so the final result is `true`

---

# Example with False

```javascript
console.log(true && false)
```

### Output

```javascript
false
```

Explanation:

- One condition is `false`    
- Therefore, the entire expression becomes `false`

---

# Logical AND Truth Table

| Expression       | Result  |
| ---------------- | ------- |
| `true && true`   | `true`  |
| `true && false`  | `false` |
| `false && true`  | `false` |
| `false && false` | `false` |

---

# Real-World Example: Driver's License Eligibility

Suppose we are building an application to determine whether a user is eligible for a driver's license.

Requirements:

- User must be older than 18    
- User must be a US citizen

---

## Eligible User

```javascript
var ageIsMoreThanEighteen = true
var isUSCitizen = true

var eligibilityForDriversLicense = ageIsMoreThanEighteen && isUSCitizen

console.log("This customer is eligible for DL: " + eligibilityForDriversLicense)
```

### Output

```javascript
This customer is eligible for DL: true
```

---

## User Under 18

```javascript
var ageIsMoreThanEighteen = false
var isUSCitizen = true

var eligibilityForDriversLicense = ageIsMoreThanEighteen && isUSCitizen

console.log("This customer is eligible for DL: " + eligibilityForDriversLicense)
```

### Output

```javascript
This customer is eligible for DL: false
```

Explanation:

- User is a US citizen    
- But user is not older than 18
- Since both conditions must be true, the result is `false`

---

## Both Conditions False

```javascript
var ageIsMoreThanEighteen = false
var isUSCitizen = false

var eligibilityForDriversLicense = ageIsMoreThanEighteen && isUSCitizen

console.log("This customer is eligible for DL: " + eligibilityForDriversLicense)
```

### Output

```javascript
This customer is eligible for DL: false
```

---

# Logical OR (`||`)

## What is Logical OR?

Logical OR works opposite to AND.

The result becomes `true` if **at least one condition is true**.

---

# Syntax

```javascript
console.log(true || false)
```

### Output

```javascript
true
```

Explanation:

- One condition is `true`    
- Therefore, the entire expression becomes `true`

---

# Both Conditions False

```javascript
console.log(false || false)
```

### Output

```javascript
false
```

Explanation:

- Both conditions are false    
- Therefore, the result is `false`

---

# Logical OR Truth Table

| Expression       | Result |
| ---------------- | ------ |
| true \|\| true   | true   |
| true \|\| false  | true   |
| false \|\| true  | true   |
| false \|\| false | false  |

---

# Driver's License Example Using OR

Suppose the requirement changes:

A user can get a driver's license if:

- They are a US citizen
- OR they are older than 18

---

## Both Conditions False

```javascript
var ageIsMoreThanEighteen = false
var isUSCitizen = false

var eligibilityForDriversLicense = ageIsMoreThanEighteen || isUSCitizen

console.log("This customer is eligible for DL: " + eligibilityForDriversLicense)
```

### Output

```javascript
This customer is eligible for DL: false
```

---

## One Condition True

```javascript
var ageIsMoreThanEighteen = false
var isUSCitizen = true

var eligibilityForDriversLicense = ageIsMoreThanEighteen || isUSCitizen

console.log("This customer is eligible for DL: " + eligibilityForDriversLicense)
```

### Output

```javascript
This customer is eligible for DL: true
```

Explanation:

- User is a US citizen
- At least one condition is true
- Therefore, the result is `true`

---

# Logical NOT (`!`)

## What is Logical NOT?

Logical NOT reverses a boolean value.

- `true` becomes `false`
- `false` becomes `true`

---

# Example

```javascript
console.log(!true)
```

### Output

```javascript
false
```

Explanation:

- NOT `true` becomes `false`

---

# Equality Example

```javascript
console.log(6 == 10)
```

### Output

```javascript
false
```

Explanation:

- `6` is not equal to `10`

---

# Not Equal Operator (`!==`)

```javascript
console.log(6 !== 10)
```

### Output

```javascript
true
```

Explanation:

- `6` is not equal to `10`
- Therefore, the result is `true`

---

# Summary

## Logical AND (`&&`)

- All conditions must be true
- Otherwise, the result is false

---

## Logical OR (`||`)

- At least one condition must be true
- Only returns false if all conditions are false

---

## Logical NOT (`!`)

- Reverses boolean values
- `true` becomes `false`
- `false` becomes `true`

---
