In this lesson, we learn about **loops** in JavaScript.

Loops are used when we want to repeat a certain operation multiple times without writing the same code repeatedly.

---

# What are Loops?

Loops help automate repetitive tasks.

Instead of writing the same code many times manually, we can use a loop to execute the code repeatedly.

---

# Example Without a Loop

Suppose we want to print `"Hello World!"` five times.

One way is to write the same statement multiple times:

```javascript
console.log("Hello World!")
console.log("Hello World!")
console.log("Hello World!")
console.log("Hello World!")
console.log("Hello World!")
```

Although this works, it is not efficient or scalable.

---

# Using a `for` Loop

The better approach is to use a loop.

---

# Basic Syntax of a `for` Loop

```javascript
for (initialStatement; condition; increment) {
	// code to execute
}
```

---

# Structure of a `for` Loop

| Part              | Purpose                           |
| ----------------- | --------------------------------- |
| Initial Statement | Starting point                    |
| Condition         | How long the loop should run      |
| Increment         | What happens after each iteration |

---

# Example: Print `"Hello World!"` Five Times

```javascript
for (let i = 0; i < 5; i++) {
	console.log("Hello World!")
}
```

### Output

```javascript
Hello World!
Hello World!
Hello World!
Hello World!
Hello World!
```

---

# Understanding the Loop

```javascript
for (let i = 0; i < 5; i++) {
	console.log("Hello World!")
}
```

### Step-by-Step

1. `let i = 0`    
    - Loop starts with `i = 0`
2. `i < 5`
    - Loop continues while `i` is less than `5`
3. `i++`
    - Increase `i` by `1` after each iteration

---

# Printing the Iterator Value

We can also print the iterator value to understand how the loop works internally.

```javascript
for (let i = 0; i < 5; i++) {
	console.log("Hello World! " + i)
}
```

### Output

```javascript
Hello World! 0
Hello World! 1
Hello World! 2
Hello World! 3
Hello World! 4
```

> [!note]  
> The loop stops before `i` becomes `5`.

# Looping Through Arrays

Loops are commonly used with arrays.

Suppose we have an array of cars:

```javascript
var cars = ["Volvo", "Toyota", "Tesla"]
```

---

# `for...of` Loop

The `for...of` loop is commonly used to iterate through array values.

```javascript
var cars = ["Volvo", "Toyota", "Tesla"]

for (let car of cars) {
	console.log(car)
}
```

### Output

```javascript
Volvo
Toyota
Tesla
```

---

# Understanding `for...of`

| Part   | Meaning              |
| ------ | -------------------- |
| `car`  | Iterator variable    |
| `cars` | Array being iterated |

During each iteration:

- `car` stores one value from the array    
- First iteration → `"Volvo"`
- Second iteration → `"Toyota"`
- Third iteration → `"Tesla"`

---

# Breaking a Loop

We can stop a loop early using the `break` statement.

---

# Example

```javascript
var cars = ["Volvo", "Toyota", "Tesla"]

for (let car of cars) {
	console.log(car)

	if (car == "Toyota") {
		break
	}
}
```

### Output

```javascript
Volvo
Toyota
```

Explanation:

- Loop stops once `"Toyota"` is found    
- `"Tesla"` is never printed

---

# `forEach()` Loop

JavaScript also provides the `forEach()` method for arrays.

This is a modern and compact way to iterate through array items.

---

# Example

```javascript
var cars = ["Volvo", "Toyota", "Tesla"]

cars.forEach(car => {
	console.log(car)
})
```

### Output

```javascript
Volvo
Toyota
Tesla
```

---

# Understanding Arrow Functions

```javascript
car => {
	console.log(car)
}
```

This is called an **arrow function**.

It is a shorter syntax introduced in ES6.

---

# Comparing Loop Types

| Loop Type   | Best Use Case            |
| ----------- | ------------------------ |
| `for` loop  | General-purpose looping  |
| `for...of`  | Iterating through arrays |
| `forEach()` | Cleaner array iteration  |

# Common Keywords

| Keyword    | Purpose                     |
| ---------- | --------------------------- |
| `break`    | Stops the loop              |
| `continue` | Skips the current iteration |
| `i++`      | Increments value by 1       |
