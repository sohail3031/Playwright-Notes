In this lesson, we learn about **conditional statements** in JavaScript.

Conditional statements allow us to control the flow of our application based on specific conditions.

---

# What are Conditional Statements?

Conditional statements are used when we want our application to make decisions.

Depending on whether a condition is `true` or `false`, different blocks of code can be executed.

---

# Basic Syntax

```javascript
if (condition) {
	// execute code if condition is true
} else {
	// execute code if condition is false
}
```

---

# How It Works

- The condition inside the parentheses `()` is evaluated.    
- If the condition is `true`, the `if` block executes.
- If the condition is `false`, the `else` block executes.

> [!note]  
> Conditions must return a boolean value (`true` or `false`).

# Real-World Example: Greeting Based on Time

Suppose we want to display different greetings depending on the hour of the day.

Requirements:

| Time Range | Message        |
| ---------- | -------------- |
| 6 to 12    | Good Morning   |
| 12 to 18   | Good Afternoon |
| Otherwise  | Good Evening   |

---

# Example 1: Hour = 5

```javascript
var hour = 5

if (hour >= 6 && hour < 12) {
	console.log("Good Morning")
} else if (hour >= 12 && hour < 18) {
	console.log("Good Afternoon")
} else {
	console.log("Good Evening")
}
```

### Output

```javascript
Good Evening
```

Explanation:

- `5` is not between `6` and `12`    
- `5` is not between `12` and `18`
- Therefore, the `else` block executes

---

# Understanding `else if`

`else if` allows us to check multiple conditions.

JavaScript evaluates conditions from top to bottom:

1. First `if`    
2. Then `else if`
3. Finally `else`

Once a condition becomes `true`, the remaining conditions are skipped.

# Driver's License Eligibility Example

We can also combine conditional statements with logical operators.

Requirements:

- User must be older than 18    
- User must be a US citizen

---

# Example 1: Eligible User

```javascript
var ageIsMoreThanEighteen = true
var isUSCitizen = true

if (ageIsMoreThanEighteen && isUSCitizen) {
	console.log("Customer is eligible for DL")
} else {
	console.log("Customer is NOT eligible for DL")
}
```

### Output

```javascript
Customer is eligible for DL
```

Explanation:

- Both conditions are true    
- Therefore, the user is eligible

---

# Example 2: Not Eligible User

```javascript
var ageIsMoreThanEighteen = true
var isUSCitizen = false

if (ageIsMoreThanEighteen && isUSCitizen) {
	console.log("Customer is eligible for DL")
} else {
	console.log("Customer is NOT eligible for DL")
}
```

### Output

```javascript
Customer is NOT eligible for DL
```

Explanation:

- User is older than 18    
- But user is not a US citizen
- Since both conditions must be true, the user is not eligible

---

# Example 3: Another Not Eligible User

```javascript
var ageIsMoreThanEighteen = false

if (ageIsMoreThanEighteen && isUSCitizen) {
	console.log("Customer is eligible for DL")
} else {
	console.log("Customer is NOT eligible for DL")
}
```

### Output


Explanation:

# Structure of Conditional Statements

| Keyword   | Purpose                                |
| --------- | -------------------------------------- |
| `if`      | Executes code when condition is true   |
| `else if` | Checks another condition               |
| `else`    | Executes when all conditions are false |

# Summary

- Conditional statements control application flow.    
- `if` executes code when a condition is true.
- `else if` checks additional conditions.
- `else` executes when no conditions match.
- Conditions usually use:
    - Relational operators
    - Equality operators
    - Logical operators
- Conditions must evaluate to `true` or `false`.
