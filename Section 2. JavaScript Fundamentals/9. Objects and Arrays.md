In this lesson, we learn about two important JavaScript data structures:

- Objects
- Arrays

These structures allow us to store multiple values inside a single variable.

---

# Objects

## What is an Object?

An object is an entity that can hold multiple values at the same time.

In JavaScript, objects store data using **key-value pairs**.

---

# Creating an Object

```javascript
var customer = {
	firstName: "John",
	lastName: "Smith"
}
```

---

## Understanding the Syntax

- Objects are defined using curly braces `{}`.    
- Data inside the object is stored as `key: value` pairs.
- Each key-value pair is separated by a comma `,`.

### Example

| Key         | Value     |
| ----------- | --------- |
| `firstName` | `"John"`  |
| `lastName`  | `"Smith"` |

---

# Printing the Entire Object

```javascript
console.log(customer)
```

### Output

```javascript
{
	firstName: "John",
	lastName: "Smith"
}
```

---

# Accessing Object Properties

There are two ways to access values inside an object:

- Dot notation    
- Bracket notation

---

# 1. Dot Notation

```javascript
console.log(customer.firstName)
console.log(customer.lastName)
```

### Output

```javascript
John
Smith
```

> [!note]  
> Dot notation is the most commonly used approach.

---

# 2. Bracket Notation

```javascript
console.log(customer["firstName"])
console.log(customer["lastName"])
```

### Output

```javascript
John
Smith
```

> [!important]  
> In bracket notation, the property name must be provided as a string.

---

# Modifying Object Properties

We can update values inside an object.

---

## Using Dot Notation

```javascript
customer.firstName = "Mike"
```

---

## Using Bracket Notation

```javascript
customer["lastName"] = "Silver"
```

---

# Updated Object

```javascript
console.log(customer.firstName)
console.log(customer.lastName)
```

### Output

```javascript
Mike
Silver
```

---

# Using Interpolation with Objects

```javascript
console.log(`${customer.firstName} ${customer.lastName}`)
```

### Output

```javascript
Mike Silver
```

---

# Arrays

## What is an Array?

An array is a collection (list) of values stored in a specific order.

Arrays are used when we want to store multiple related items.

---

# Creating an Array

```javascript
var cars = ["Volvo", "Toyota", "Tesla"]
```

---

# Understanding Arrays

- Arrays are defined using square brackets `[]`    
- Values are separated using commas
- Each item has an index
- Array indexing starts from `0`

---

# Array Indexes

| Value      | Index |
| ---------- | ----- |
| `"Volvo"`  | `0`   |
| `"Toyota"` | `1`   |
| `"Tesla"`  | `2`   |

---

# Accessing Array Values

```javascript
console.log(cars[0])
```

### Output

```javascript
Volvo
```

---

# Accessing Other Elements

```javascript
console.log(cars[1])
console.log(cars[2])
```

### Output

```javascript
Toyota
Tesla
```

---

# Modifying Array Values

We can replace values inside an array using their index.

```javascript
cars[1] = "BMW"

console.log(cars[1])
```

### Output

```javascript
BMW
```

---

# Arrays Inside Objects

Objects can also contain arrays.

---

## Example

```javascript
var customer = {
	firstName: "John",
	lastName: "Smith",
	cars: ["Volvo", "Toyota", "Tesla"]
}
```

---

# Accessing Array Values Inside an Object

```javascript
console.log(customer.cars[0])
```

### Output

```javascript
Volvo
```

---

# Objects vs Arrays

| Objects                       | Arrays                        |
| ----------------------------- | ----------------------------- |
| Store data as key-value pairs | Store ordered lists of values |
| Defined using `{}`            | Defined using `[]`            |
| Access values using keys      | Access values using indexes   |
| Example: `customer.firstName` | Example: `cars[0]`            |

---

# Important Notes

## Objects

- Use objects when data has meaningful property names.    
- Objects are flexible and can store different data types.

---

## Arrays

- Arrays maintain order.    
- Indexes always start from `0`.
- Arrays are useful for storing lists of items.

---

# Summary

- Objects store data using key-value pairs.    
- Arrays store ordered lists of values.
- Object properties can be accessed using:
    - Dot notation
    - Bracket notation
- Arrays use indexes to access elements.
- Objects can contain arrays.
- Arrays and objects together help create complex data structures in JavaScript.

> [!note]  
> Arrays and objects are foundational concepts in JavaScript and are heavily used in web development and Playwright automation.
