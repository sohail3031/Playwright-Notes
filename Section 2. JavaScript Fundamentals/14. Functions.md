In this lesson, we learn about **functions** in JavaScript.

Functions are one of the most important concepts in programming because they help us organize, reuse, and simplify code.

---

# What is a Function?

A function is a reusable block of code that performs a specific task.

Instead of writing the same code multiple times, we can place the code inside a function and call it whenever needed.

---

# Why Do We Need Functions?

Functions help us:

- Reuse code
- Avoid repetition
- Organize logic
- Improve readability
- Make applications easier to maintain

---

# Example Without a Function

Suppose we want to print `"Hello World!"` multiple times.

```javascript
console.log("Hello World!")
console.log("Hello World!")
console.log("Hello World!")
```

This works, but repeating code is not efficient.

---

# Creating a Function

## Basic Syntax

```javascript
function functionName() {
	// code to execute
}
```

---

# Example: Simple Function

```javascript
function greet() {
	console.log("Hello World!")
}
```

---

# Calling a Function

A function does not execute automatically after creation.

We must call it.

```javascript
greet()
```

### Output

```javascript
Hello World!
```

---

# Calling a Function Multiple Times

```javascript
greet()
greet()
greet()
```

### Output

```javascript
Hello World!
Hello World!
Hello World!
```

---

# Function Parameters

Functions can accept input values called **parameters**.

Parameters allow functions to work with dynamic data.

---

# Example

```javascript
function greetUser(name) {
	console.log("Hello " + name)
}
```

---

# Calling the Function with Arguments

```javascript
greetUser("John")
greetUser("Mike")
```

### Output

```javascript
Hello John
Hello Mike
```

---

# Understanding Parameters and Arguments

| Term      | Meaning                             |
| --------- | ----------------------------------- |
| Parameter | Variable inside function definition |
| Argument  | Actual value passed to function     |

Example:

```javascript
function greetUser(name) {
	console.log(name)
}

greetUser("John")
```

- `name` → parameter    
- `"John"` → argument

---

# Multiple Parameters

Functions can accept multiple parameters.

```javascript
function printFullName(firstName, lastName) {
	console.log(firstName + " " + lastName)
}
```

---

# Calling the Function

```javascript
printFullName("John", "Smith")
```

### Output

```javascript
John Smith
```

---

# Returning Values from Functions

Functions can also return values using the `return` keyword.

---

# Example

```javascript
function add(a, b) {
	return a + b
}
```

---

# Using the Returned Value

```javascript
var result = add(5, 3)

console.log(result)
```

### Output

```javascript
8
```

---

# Understanding `return`

The `return` statement:

- Sends a value back from the function    
- Stops function execution immediately

---

# Function Scope

Variables declared inside a function exist only inside that function.

---

# Example

```javascript
function testScope() {
	var message = "Hello"
	console.log(message)
}

testScope()
```

### Output

```javascript
Hello
```

---

# Arrow Functions (ES6)

JavaScript also provides a shorter function syntax called **arrow functions**.

---

# Traditional Function

```javascript
function sayHello(name) {
	console.log("Hello " + name)
}
```

---

# Arrow Function Version

```javascript
const sayHello = (name) => {
	console.log("Hello " + name)
}
```

---

# Single-Line Arrow Function

```javascript
const multiply = (a, b) => a * b
```

---

# Calling Arrow Functions

```javascript
console.log(multiply(2, 3))
```

### Output

```javascript
6
```

---

# Types of Functions

| Function Type            | Description                     |
| ------------------------ | ------------------------------- |
| Regular Function         | Traditional JavaScript function |
| Function with Parameters | Accepts input values            |
| Function with Return     | Returns a value                 |
| Arrow Function           | Modern ES6 shorthand syntax     |
