## Introduction

In this lesson, we’ll learn how to create **parameterized methods** inside Playwright Page Objects.

We’ll build methods for the **Form Layouts** page that can:

- Fill and submit the **Using the Grid** form    
- Fill and submit the **Inline form**
- Accept **dynamic input values** through parameters
- Improve **reusability** of the Page Object methods

Parameterized methods are useful because they let you reuse the same method in multiple tests with different test data.

---

## Goal of This Lesson

We will create a new page object called `FormLayoutPage` and add methods for:

1. **Submitting the "Using the Grid" form**    
    - Email
    - Password
    - Radio option
2. **Submitting the "Inline form"**
    - Name
    - Email
    - Remember Me checkbox

---

import { Page } from "@playwright/test";

export class FormLayoutPage {
  private readonly page: Page;

  }
}

# 1) Parameterized Method for "Using the Grid" Form

We’ll create a method that accepts:

- `email: string`    
- `password: string`
- `optionText: string`

This method will:

- Fill the email field
- Fill the password field
- Select the specified radio option
- Click the submit button

## Implementation

```typescript
// formLayoutsPage.ts
import { Page } from "@playwright/test";

export class FormLayoutPage {
  private readonly page: Page;

  constructor(page: Page) {
    this.page = page;
  }

  async submitUsingTheGridFormWithCredentialsAndSelectOption(
    email: string,
    password: string,
    optionText: string
  ) {
    const usingTheGridForm = this.page.locator("nb-card", { hasText: "Using the Grid" });

    await usingTheGridForm.getByRole("textbox", { name: "Email" }).fill(email);
    await usingTheGridForm.getByRole("textbox", { name: "Password" }).fill(password);
    await usingTheGridForm.getByRole("radio", { name: optionText }).check({ force: true });
    await usingTheGridForm.getByRole("button").click();
  }
}
```

# Using the Page Object in a Test

test("Parameterized Methods", async ({ page }) => {

# Using Different Data with the Same Method

One of the biggest advantages of parameterized methods is that you can reuse the same implementation with different input values.

For example, to select **Option 2** instead of **Option 1**:

```typescript
test("Parameterized Methods", async ({ page }) => {
  const navigateTo = new NavigationPage(page);
  const onFormLayoutsPage = new FormLayoutPage(page);

  await navigateTo.formLayoutsPage();
  await onFormLayoutsPage.submitUsingTheGridFormWithCredentialsAndSelectOption(
    "test@test.com",
    "Welcome1",
    "Option 2"
  );
});
```

# 2) Parameterized Method for the Inline Form

Now let’s create another method for the **Inline form**.

This form accepts:

- `name: string`    
- `email: string`
- `rememberMe: boolean`

The boolean parameter will decide whether the **Remember Me** checkbox should be checked.

---

## Implementation

```typescript
// formLayoutsPage.ts
import { Page } from "@playwright/test";

export class FormLayoutPage {
  private readonly page: Page;

  constructor(page: Page) {
    this.page = page;
  }

  async submitUsingTheGridFormWithCredentialsAndSelectOption(
    email: string,
    password: string,
    optionText: string
  ) {
    const usingTheGridForm = this.page.locator("nb-card", { hasText: "Using the Grid" });

    await usingTheGridForm.getByRole("textbox", { name: "Email" }).fill(email);
    await usingTheGridForm.getByRole("textbox", { name: "Password" }).fill(password);
    await usingTheGridForm.getByRole("radio", { name: optionText }).check({ force: true });
    await usingTheGridForm.getByRole("button").click();
  }

  async submitInlineFormWithNameEmailAndCheckbox(
    name: string,
    email: string,
    rememberMe: boolean
  ) {
    const inlineForm = this.page.locator("nb-card", { hasText: "Inline form" });

    await inlineForm.getByRole("textbox", { name: "Jane Doe" }).fill(name);
    await inlineForm.getByRole("textbox", { name: "Email" }).fill(email);

    if (rememberMe) {
      await inlineForm.getByRole("checkbox").check({ force: true });
    }

    await inlineForm.getByRole("button").click();
  }
}
```

---

# Calling Both Methods in a Test

```typescript
test("Parameterized Methods", async ({ page }) => {
  const navigateTo = new NavigationPage(page);
  const onFormLayoutsPage = new FormLayoutPage(page);

  await navigateTo.formLayoutsPage();

  await onFormLayoutsPage.submitUsingTheGridFormWithCredentialsAndSelectOption(
    "test@test.com",
    "Welcome1",
    "Option 2"
  );

  await onFormLayoutsPage.submitInlineFormWithNameEmailAndCheckbox(
    "John Smith",
    "john@test.com",
    true
  );
});
```

---

# Checkbox Behavior with Boolean Parameters

If you pass `true`, the checkbox is checked:

```typescript
await onFormLayoutsPage.submitInlineFormWithNameEmailAndCheckbox(
  "John Smith",
  "john@test.com",
  true
);
```

If you pass `false`, the checkbox is **not** checked:

```typescript
await onFormLayoutsPage.submitInlineFormWithNameEmailAndCheckbox(
  "John Smith",
  "john@test.com",
  false
);
```

This is a great example of how parameterized methods make your test flows flexible without duplicating code.

# Adding Documentation with JSDoc

You can make your Page Object methods easier to understand by adding **JSDoc comments**.

For example:

```typescript
import { Page } from "@playwright/test";

export class FormLayoutPage {
  private readonly page: Page;

  constructor(page: Page) {
    this.page = page;
  }

  async submitUsingTheGridFormWithCredentialsAndSelectOption(
    email: string,
    password: string,
    optionText: string
  ) {
    const usingTheGridForm = this.page.locator("nb-card", { hasText: "Using the Grid" });

    await usingTheGridForm.getByRole("textbox", { name: "Email" }).fill(email);
    await usingTheGridForm.getByRole("textbox", { name: "Password" }).fill(password);
    await usingTheGridForm.getByRole("radio", { name: optionText }).check({ force: true });
    await usingTheGridForm.getByRole("button").click();
  }

  /**
   * Fills out the Inline form with user details.
   *
   * @param name - First and last name of the user
   * @param email - Valid email address for the test user
   * @param rememberMe - Whether the Remember Me checkbox should be selected
   */
  async submitInlineFormWithNameEmailAndCheckbox(
    name: string,
    email: string,
    rememberMe: boolean
  ) {
    const inlineForm = this.page.locator("nb-card", { hasText: "Inline form" });

    await inlineForm.getByRole("textbox", { name: "Jane Doe" }).fill(name);
    await inlineForm.getByRole("textbox", { name: "Email" }).fill(email);

    if (rememberMe) {
      await inlineForm.getByRole("checkbox").check({ force: true });
    }

    await inlineForm.getByRole("button").click();
  }
}
```

---

# Why JSDoc Is Useful

When you hover over the method in VS Code, you’ll see:

- What the method does    
- What each parameter means
- How the method should be used

This makes Page Objects easier to navigate and maintain, especially in larger automation frameworks.
